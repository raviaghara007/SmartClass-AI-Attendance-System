# SmartClass AI Attendance System
An AI-powered attendance system using Face Recognition and Voice Recognition.


## Live Demo

Frontend
https://smartclass-ai-attendance-system.vercel.app/

Backend
https://smartclass-maian.streamlit.app/


## Features
- Face Recognition Attendance
- Voice Recognition Attendance
- QR Code Subject Enrollment
- Teacher Dashboard
- Student Dashboard
- Subject Management
- Attendance Reports

Backend
